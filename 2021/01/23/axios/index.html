源码分析

  1. 从语法上说 axios 不是 Axios 的实例
  2. 从功能上说 axios 是 Axios 的实例
    1. axios 通过 utils.extend 复制了 Axios 的函数和属性
    2. axios 仍旧保持是一个函数所以可以直接 axios()使用
  3. axios 是 Axios.prototype.request 函数 bind()函数返回的函数
